# HomeStock AI — Flask

## Estructura
```
homestockai/
├── app.py            ← Backend Flask (lógica + API REST)
├── data.json         ← Se crea automáticamente al primer arranque
├── requirements.txt
└── templates/
    └── index.html    ← Frontend (HTML/CSS/JS)
```

## Cómo correr

### 1. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 2. Correr la app
```bash
python app.py
```

### 3. Abrir en el navegador
```
http://localhost:5000
```

## API REST

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | /api/products | Lista todos los productos (con campos calculados) |
| POST | /api/products | Crear nuevo producto |
| PUT | /api/products/<id> | Editar producto |
| DELETE | /api/products/<id> | Eliminar producto |
| POST | /api/products/<id>/entries | Agregar entrada (compra) |
| DELETE | /api/products/<id>/entries/<eid> | Eliminar entrada |
| PATCH | /api/products/<id>/close | Cerrar entrada abierta ("Terminé esto") |
| POST | /api/mercado | Registrar mercado completo |
| GET | /api/charts | Datos para gráficas |
| GET | /api/lista-mercado | Lista de productos a comprar pronto |

## Lógica de IA

- **n < 5 entradas**: promedio simple de tasas históricas
- **n ≥ 5 entradas**: suavizamiento exponencial (ETS, alpha=0.3) — más peso al consumo reciente
- Clasificación: Crítico ≤3d / Atención ≤7d / OK >7d

## Datos

Los datos se guardan en `data.json`. La primera vez se cargan automáticamente
los datos históricos reales del hogar (agosto 2025 – abril 2026).
Para reiniciar, borra el archivo `data.json`.
